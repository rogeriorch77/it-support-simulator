# Contributing to IT Support Simulator

Thanks for taking the time to help improve this project! 🙌

---

## How to Report a Bug

1. Go to the **[Issues](../../issues)** tab at the top of this repository
2. Click **"New issue"**
3. Choose **"Bug report"** (or write a title that starts with `[Bug]`)
4. Describe the problem clearly:
   - What were you doing when the bug happened?
   - What did you expect to happen?
   - What actually happened?
   - Which browser / device are you using?
5. Click **"Submit new issue"**

> **Example title:** `[Bug] Simulation freezes after selecting correct answer`

---

## How to Suggest a Feature

1. Go to the **[Issues](../../issues)** tab
2. Click **"New issue"**
3. Start the title with `[Feature]`
4. Describe your idea:
   - What would you like to see?
   - Why would it be useful?
5. Click **"Submit new issue"**

> **Example title:** `[Feature] Add a daily challenge mode to the quiz`

---

## General Guidelines

- Search existing issues before opening a new one — it may already be reported
- Be specific and concise — the more detail, the faster it gets fixed
- Screenshots are always welcome
- Be respectful and constructive

---

## What Can You Contribute?

| Type | Examples |
|------|---------|
| 🐛 Bug report | Something broken, freezing, not displaying correctly |
| 💡 Feature request | New module, new scenario, UX improvement |
| 📝 Content | Spelling mistakes, incorrect answers in quiz/scenarios |
